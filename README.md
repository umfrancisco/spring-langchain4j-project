# Spring Boot Local Text Generation with LangChain4j & Ollama

This repository contains a Spring Boot 4 application demonstrating how to build a local, private, and cost-free text generation pipeline using LangChain4j and Ollama.

## Tech Stack

- Java 17+
- Maven 3.8+
- LangChain4J
- Ollama
